PDFSAM@What's new

PDFsam v3 M1 released!

13th Dec 2014

PDFsam 3.0.0.M1 is ready for tests. The missing workspace feature has been implemented together with some bug fixes and user suggestions. Here is a complete list of what's in this release. Feel free to contribute bug reports or suggestions and don't forget, if you want to translate PDFsam to your language, this is the place.

PDFsam v3 first beta ready to test!

26th Oct 2014

After an enormous effort, the first BETA version of PDFsam v3 is finally available. It is not yet feature complete but very little is missing (i.e. workspace related features, platform specific bundles). Check out the project page to report an issue or make a suggestion and for the braves, fork the code and start contributing!
